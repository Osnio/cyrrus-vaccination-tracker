# VacinaKids

Aplicação web para acompanhamento da vacinação infantil, permitindo visualizar o histórico vacinal, identificar vacinas pendentes e acompanhar campanhas de vacinação através de uma experiência simples e intuitiva.

## Funcionalidades

* Dashboard com indicadores vacinais
* Gestão de múltiplas crianças
* Histórico e timeline de vacinação
* Identificação de vacinas aplicadas, pendentes e atrasadas
* Catálogo de vacinas
* Campanhas de vacinação
* Relatórios e métricas de acompanhamento

## Como Executar

```bash
git clone https://github.com/Osnio/cyrrus-vaccination-tracker.git

cd cyrrus-vaccination-tracker

npm install

npm start
```

A aplicação estará disponível em:

```bash
http://localhost:4200
```
